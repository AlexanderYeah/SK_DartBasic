3 变量

Dart 类型是安全的，使用静态类型检查和运行时的组合。

在Dart 中，变量必须是明确的类型或者是系统能够解析的类型。如下

```dart

var name = "dog";
print(name);

String cls  = "Dod";
print(cls);
```



4 默认值，在Dart中，未初始化的变量的初始值为null

```dart
var name;
// null
print(name);
```



5 方法声明与调用

```dart
run(){
    print("run");
}
run();
```
